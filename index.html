IcySeq is an extremely simple nucleotide (only) sequence alignment viewer, intended for making rapid and superficial assessments of alignment files. It reads most FASTA files and some NEXUS files, but be warned that the NEXUS parser is extremely flimsy. To report issues, request features or to fork the project, visit its GitHub page.
